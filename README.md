# CV Repositioning Strategy

## 1. Broaden the Target Roles

Current profile should not be positioned only toward narrow RF/RAN research roles.

Target broader engineering categories including:
- Systems Engineering/ Analyst
- Systems Integration
- Validation / QA Engineering
- Technical Support Engineering
- Deployment Engineering
- Telecom Systems
- Infrastructure / Networking
- IoT Systems
- Platform / Operations Engineering

Core realization:
Most technical work performed was not purely “RF work,” but rather:
- complex systems debugging
- integration
- validation
- automation
- infrastructure troubleshooting
- performance analysis
- hardware/software/network interaction

Wireless becomes a specialization, not the entire identity.

---

## 2. Reduce Excessive Jargon

Keep technical depth while making the CV understandable to:
- recruiters
- ATS systems
- non-telecom hiring managers

Avoid overloading bullets with highly niche terminology unless directly relevant to the target role.

Examples of problematic heavy jargon:
- CQI
- MCS
- PMK caching
- NR-U
- PDU session
- AMF/SMF/UPF
- QMI

Translate technical work into broadly understandable engineering language:
- systems debugging
- protocol troubleshooting
- infrastructure validation
- network analysis
- integration testing
- performance characterization
- reliability improvement

Goal:
make the engineering value immediately understandable within seconds.

---

## 3. Add a Clear Professional Hook

Add a concise positioning statement under the name/header.

Examples:
- Systems Integration & Wireless Validation Engineer
- Systems, Networking & Wireless Validation Engineer
- Integration, Validation & Wireless Systems Engineer

Optional 1-line summary:
“Hands-on experience in systems integration, wireless validation, Linux networking, SDR-based 5G testing, and infrastructure troubleshooting.”

Purpose:
- immediately categorize the profile
- reduce recruiter confusion
- avoid “too academic” perception
- improve first-pass readability

Important:
- keep it short
- no motivational fluff
- no large summaries
- no buzzword paragraphs
